# Windmill-Pitch-System-IoT-Project
---
## Project Description
This project is in collaboration with CSULB ASME's Windmill Project within its Electrical Team. Specifically, this repo will cover controls, IoT, and Data Acquisition, monitoring the windmill's health, power generation, and custom pitch system.
